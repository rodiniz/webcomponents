# Web Components Skills

AI agent skills for the @your-org/webcomponents library.

## Installation

```bash
npx skills add your-org/webcomponents-skills
```

## Usage

The skill will be automatically available when working with the web components library. The AI agent will use these instructions when you:

- Create or modify ui-button, ui-input, ui-table, ui-date-picker, or ui-pagination components
- Add validation to forms
- Set up data tables
- Work with pagination

## Components

- **ui-button** - Button with variants, sizes, and icons
- **ui-input** - Input with native and custom validation
- **ui-table** - Data table with actions
- **ui-date-picker** - Date picker component
- **ui-pagination** - Pagination control
