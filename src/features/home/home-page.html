Why Use This Library?

✨

Native Web Components

Built on standard Custom Elements API. Works with any framework or none at all.

⚡

Reactive Signals

Built-in signal-based reactivity system similar to Solid.js for smooth updates.

🎨

Theme Support

CSS custom properties make theming and customization effortless.

📦

Zero Dependencies

No framework required. Lightweight and fast with nothing to bloat your bundle.

🔒

TypeScript Support

Full type safety and IntelliSense support for better development experience.

♿

Accessible

Built with ARIA attributes and keyboard navigation from the ground up.

Available Components

ui-button Flexible button with variants and sizes

ui-table Data table with sorting and actions
ui-modal Dialog modal component
ui-tabs Tab navigation component

ui-toast Toast notification system
ui-stepper Multi-step progress indicator
ui-upload File upload with drag-drop
ui-layout Flexible page layout system

Quick Start

Installation

npm install @diniz/webcomponents

Basic Usage

<ui-button variant="primary">Click Me</ui-button>
<ui-input placeholder="Enter text"></ui-input>
<ui-date-picker format="MM/DD/YYYY"></ui-date-picker>

With Events

const picker = document.querySelector('ui-date-picker');
picker?.addEventListener('date-change', (e) => {
  console.log('Date:', e.detail.value);
});
